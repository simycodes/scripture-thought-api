# scripture-thought-api
There are a few online platforms that are providing a way of sharing personal scriptural messages or thoughts with others of the same faith and spirituality. Hence, Scripture Thought API is intended to provide a solid API platform for such a platform that will allow users to share their spiritual thoughts with others.
